# 接口文档

## 用户模块接口设计（RESTful）

| 功能                 | 方法   | 路径       |
| -------------------- | ------ | ---------- |
| 获取用户列表（分页） | GET    | /users     |
| 获取用户详情         | GET    | /users/:id |
| 创建用户             | POST   | /users     |
| 更新用户             | PUT    | /users/:id |
| 删除用户             | DELETE | /users/:id |

## 角色管理

角色页面只负责三件事：

- 角色增删改查（Role CRUD）
- 给角色分配菜单权限（role_menu）
- （可选）查看角色权限预览
