# React Admin Monorepo

## 📌 项目简介

这是一个基于 React + TypeScript + Monorepo 的后台管理系统 Demo 项目。

包含：

- admin-web：前端后台系统
- admin-server：Node.js API 服务
- shared：公共工具库

## 🚀 技术栈

- React
- TypeScript
- Ant Design
- Vite
- Node.js
- JWT
- pnpm workspace

## 📦 目标

用于练习：

- 前端工程化
- 中后台系统架构设计
- 登录鉴权（JWT）
- Dashboard 数据可视化
